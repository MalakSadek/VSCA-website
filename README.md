
# VSCA Framework - Websites

## Usage


1. Clone this Repository
2. Navigate to the directory
3. Install `npm install`
4. Run locally: `npm run dev`
5. Deploy to verccel


## Colour Scheme

* Primary: #276197
* Secondary: #47A9DC
* Accent: #EEA947
